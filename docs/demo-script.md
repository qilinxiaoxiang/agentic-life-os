# 60–90 second product walkthrough

## Setup

- Run the synthetic demo with `docker compose up --build`.
- Keep the Portal open on Today and a terminal beside it.
- Use the included money and time batch examples.

## Script

**0–12s — Thesis**

“Most personal systems keep tasks, time, and money apart. Agentic Life OS gives
an AI agent one local control surface for all three—without giving it silent
authority over your ledgers.”

Show the Today tab, daily focus, and ranked actionables.

**12–28s — Observe and act**

Run `lifeos context`, then add one task. Say: “The agent reads the real state
first. Reversible work, like a task, can move immediately.” Refresh the page.

**28–48s — Budget both scarce resources**

Open Time, then Money. Say: “Time is a 168-hour budget with separate physical
and allocation totals. Money compares accounts, monthly plans, and actuals.”

**48–70s — Human authority**

Run both preview commands. Show the pending cards. Say: “For durable time and
money records, the agent can only propose. Nothing changes until I inspect and
confirm the normalized batch.” Click Confirm & commit and show the update.

**70–85s — Close**

“It is local-first, model-independent, and open source. Bring Codex, Claude,
OpenClaw, or any agent that can call HTTP or a CLI. Your agent proposes; you
remain the authority.”
